## Что это значит для инвестора

CAGR (7.86%) отражает среднегодовой темп роста по месячным простым доходностям на 10Y-окне в текущем прогоне. Волатильность (8.00%) — годовая из месячных доходностей; MaxDD (-14.03%) — по месячной equity-кривой. Sharpe (0.709) и Sortino (1.138) используют спецификацию проекта (знаменатель — vol сырой доходности для Sharpe). чувствительность к широкому рынку (0.403) и оценка доходности с учётом чувствительности к рынку (0.141) завязаны на базовый бенчмарк; связь с рынком в целом при наличии показывает синхронность с бенчмарком на том же окне.

**Сильные стороны.**

Диагностический стресс без критичных отметок (или только предупреждения); мандатный MaxDD-gate в норме по проверке — сочетание исторической просадки и клиентского порога не конфликтует в этом прогоне.
