# Анализ доходности ETF VOO

Этот скрипт загружает данные по ETF VOO за последние 10 лет из Yahoo Finance и вычисляет среднюю годовую доходность.

## Установка зависимостей

```bash
pip install yfinance pandas
```

или

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python voo_analysis.py
```

## Что делает скрипт

1. Загружает исторические данные по VOO за последние 10 лет из Yahoo Finance
2. Вычисляет:
   - Общую доходность за весь период
   - CAGR (Compound Annual Growth Rate) - средняя годовая доходность с учетом сложного процента
   - Среднюю годовую доходность (простое среднее по годам)
   - Детальную доходность по каждому году
3. Сохраняет результаты в файл `voo_annual_returns.csv`

## Результаты

Скрипт выводит:
- Период анализа
- Начальную и конечную цены
- Общую доходность за период
- CAGR (рекомендуемый показатель средней годовой доходности)
- Среднюю годовую доходность (простое среднее)
- Таблицу доходности по годам
